Takes about 30 seconds each time you add a clip.
  1. 1
    Go to Add a Clip → complete all steps → copy the generated code
  2. 2
    Open player.html in any text editor → find const CLIPS = {

    Paste your code inside the { brackets → Save the file
  4. 4
    Upload the updated player.html to GitHub — see next step
3
Upload to GitHub
Takes about one minute. No coding needed.
  1. 1
    Go to your GitHub repo → Add file → Upload files
  2. 2
    Drag in your updated player.html, clip files into clips/, images into images/
  3. 3
    Click Commit changes — your player goes live within 30 seconds

Create your GitHub repository
GitHub hosts your player and clips online, for free. Takes about 3 minutes.
  1. 1
    Go to github.com → sign in or create a free account
  2. 2
    Click the ＋ (top right) → New repository
  3. 3
    Name it albert-s-leung → set to Public → click Create repository
  4. 4
    Go to Settings → Pages → Source: Deploy from a branch → main → / (root) → Save
2
Upload the player files
Drag and drop — no coding at all.
  1. 1
    In your new repo → Add file → Upload files
  2. 2
    Unzip the downloaded package. Drag the entire folder's contents into GitHub
  3. 3
    Click Commit changes — your site goes live in 30–60 seconds
